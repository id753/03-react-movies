# React + TypeScript + Vite
# Сервіс пошуку фільмів TMDB</br>
 Застосунок пошуку фільмів за ключовим словом.</br>

Для виконання HTTP-запитів використовується бібліотека axios. </br>
Для сповіщень використовуйте бібліотеку React Hot Toast.</br>
Хедер з формою пошуку SearchBar</br>
Модальне вікно має створюватись через createPortal</br>
Під час натискання на зображення галереї повинно відкриватися модальне вікно, яке відображатиме додаткову інформацію про фільм у великому форматі. Створіть для цього компонент MovieModal.</br>
Компонент MovieGrid – це список карток фільмів. </br>
Повідомлення про помилку ErrorMessage</br>
Індикатор завантаження Loader</br>
Для типізації пропсів компонентів використовується interface.</br>
